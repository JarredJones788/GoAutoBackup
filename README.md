
GoAutoBackup

- Supports remote repo's
- Supports Folders / Files
- Supports MySql

CLI Service Commands
- autobackup init (First time use, creates a systemd service)
- autobackup start 
- autobackup stop
- autobackup restart
- autobackup list
- autobackup add
- autobackup remove [Tag Name]
- autobackup status
- autobackup disable
